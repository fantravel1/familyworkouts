## Family Workout Card Format

Each card includes:

- **Time:** 3 / 5 / 8 / 12 minutes  
- **Space:** 1 small room / hallway / backyard / park  
- **Ages:** 2–4 / 5–8 / 9–12 / teens / adults / all  
- **Intensity knobs:** Easy / Medium / Spicy (same game)  
- **Accessibility mods:** chair version / no jumping / quiet mode  
- **Goal:** cardio / strength / mobility / coordination / balance  
- **Family roles:** kid leads / parent leads / rotate leader  
- **Safety cues:** simple “stop rules” + form hints  
- **Fun hook:** the story / game / mission  

This is the core moat.
